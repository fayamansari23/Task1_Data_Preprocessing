# Task 1 – Data Cleaning & Preprocessing

This repository contains the work for Task 1 of the internship.

## Files included:
- **Task1_Data_Preprocessing.ipynb** – Complete Jupyter Notebook
- **cleaned_titanic.csv** (you will generate this by running the notebook)
- **titanic.csv** (upload your dataset here)

## Steps Performed:
1. Loaded Titanic dataset
2. Checked missing values and data info
3. Filled missing Age with median and Embarked with mode
4. Dropped Cabin due to many nulls
5. Encoded categorical features
6. Visualized outliers using boxplots
7. Removed outliers using IQR method
8. Scaled Age and Fare using StandardScaler
9. Exported cleaned dataset

Make sure to upload your dataset file (**titanic.csv**) before running the notebook.
